# Ecological Modeling Project
This repository contains a semester-long ecological modeling project completed as coursework. The project analyzes ecological data using statistical models in R.

## Research Goal
The objective of this project was to explore relationships between ecological variables and identify patterns using statistical modeling techniques.

## Methods
The analysis was performed using R and RMarkdown. The workflow included:

* data cleaning and preparation
* exploratory data analysis
* statistical modeling
* visualization of ecological relationships

## Repository Structure
code/ – RMarkdown file containing the full analysis
paper/ – final written report for the project

## Tools Used
* R
* RMarkdown
* data visualization
* statistical modeling
